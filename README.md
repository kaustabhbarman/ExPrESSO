# Extensive Privacy Preserving Single Sign-On (ExPrESSO)

## Getting Started


### Prerequisites
- Python 3.9 or higher
- [Pipenv](https://pipenv.pypa.io/en/latest/)
- [ZoKrates](https://zokrates.github.io/gettingstarted.html#installation)

### Installation 
```bash
git clone <this repo>
cd ExPrESSO
```

Install dependencies:
```shell
pipenv install
pipenv install --dev
```

## Run 

```bash
$ pipenv run dev
```
